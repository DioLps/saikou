Saikou api - Endpoints List 😏

Este Api é um crawler que foi feito com base em um dos melhores sites hoje para ver animes, goyabu.com. A ideia por trás do desenvolvimento da mesma, foi estudar o funcionamento dos mecânismos de busca e indexação de conteúdo. Projeto sem fins lucrativos.
  • Index - Você está aqui :)
  • Animes
  • Detalhes - Necessário hash pego no endpoint de animes, estou usando o 30 como exemplo
  • Episódios - Necessário slug pego no endpoint de animes e o número da página, estou usando o black-clover e a página 1 como exemplo
  • Vídeos - Necessário slug pego no endpoint de episódios, estou usando o 72149 como exemplo
